Maven dependencies

Jhades can be installed with this maven dependency:


<dependency>
    <groupId>org.jhades</groupId>
    <artifactId>jhades</artifactId>
    <version>1.0.4</version>
</dependency>

The WAR duplicate classes report can be installed with this dependency:


<dependency>
    <groupId>org.jhades</groupId>
    <artifactId>jhades-standalone-report</artifactId>
    <version>1.0.4</version>
</dependency>
